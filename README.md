This platform serves as a Zoho Creator replacement for managing TopFly and Jointech hardware.
- Multi-tenant Architecture: Supports administrators, resellers, and multiple companies.
- Device Management: View, group, and filter devices and cards.
- Lifecycle Operations: Insert, replace, or remove cards from devices.
- Asynchronous Sync: Requests current card assignments from devices and monitors results via background processes.
